\begin{abstract}
在复杂多变的市场环境中，寻找具备防御属性且能提供稳定现金流的投资标的成为资产配置的重要课题。港股市场作为包含大量低估值、高股息特征公司的市场，天然适合运用红利类策略。

本报告基于\textbf{恒生综合指数（HSCI）}作为基础股票池，循序渐进地构建并对比了三种策略：单一回购因子策略（v1）、高红利低波动策略（v2）以及综合收益率（Shareholder Yield）策略（v3）。
\end{abstract}

\section{研究背景与逻辑概述}

本报告主要基于\textbf{恒生综合指数（HSCI）}作为基础股票池，构建并对比了以下三种策略：
\begin{enumerate}[label=1.\arabic*]
    \item \textbf{单一回购因子策略（v1）}：侧重挖掘持续以真金白银回购公司股份的标的。
    \item \textbf{高红利低波动策略（v2）}：结合股息率与低波动率，进一步排除高波动个股。
    \item \textbf{综合收益率（Shareholder Yield）策略（v3）}：将股息收益与回购收益相结合，并叠加低波过滤。
\end{enumerate}

\section{策略完整构建方案}

以下为最终进阶版——\textbf{综合收益率（Shareholder Yield）策略}的完整构建逻辑：

\subsection{股票池与基础清洗}
\begin{itemize}
    \item \textbf{基础池}：恒生综合指数（HSCI）成分股。
    \item \textbf{基本面过滤}：过去 36 个月有分红或回购记录。
    \item \textbf{流动性与市值过滤}：剔除后 20\%（仅保留 ADTV 和市值前 80\%）。
\end{itemize}

\subsection{核心排序因子：综合收益率}
\begin{align*}
\text{DivYield}_i &= \dfrac{\text{3-Year Avg DPS}}{\text{Current Price}} \\
\text{BuybackYield}_i &= \dfrac{\text{Total Buyback in 3Y}/3}{\text{Current Market Cap}} \\
\text{ShareholderYield}_i &= \text{DivYield}_i + \text{BuybackYield}_i
\end{align*}

\subsection{极端值剔除与低波优选}
\begin{itemize}
    \item 剔除前 0.1\% 极端高收益率样本。
    \item 选取因子值前 20 - 35\% (基于候选池大小) 进入备选池。
    \item 选取波动率最低的 15～20 只股票作为最终持仓。
\end{itemize}

\subsection{资产配置与调仓}
等权配置 或 受限市值加权（单股上限 10\%），定期季度调仓。

\section{报告总结与超额 Alpha 深度解读}

v3 综合收益率策略在年化收益率、夏普比率和风险控制上均表现最优。其核心优势在于将股息与回购结合，并通过低波动筛选实现了高质量的阿尔法来源。

Fama-French 三因子回归显示 Q5 组每月 Alpha 高达 0.521\%（t=4.45），多空组合 Alpha 也极为显著，证明“股东现金回报溢价”是一个独立且强有力的选股因子。

\textbf{最终结论}：综合收益率（分红+回购）+低波动策略是港股市场中兼具防御性与超额收益潜力的优秀量化策略，适合长期稳健配置。
